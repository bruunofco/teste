## Guia Rápido de Indicação

### Para Jardim Residencial
- **Econômico**: BrassSoft Prime 20mm ou 30mm (R$ 89-129/m²)
- **Intermediário**: BrassSoft Max 30mm ou BrassGold 30mm (R$ 149/m²)
- **Premium**: BrassSoft Luxo 30mm ou 40mm (R$ 230-240/m²)

### Para Área Pet
- **Recomendado**: BrassSoft 20mm (4m) ou BrassSoft Prime 20mm
- **Motivo**: Resiste ao pisoteio, fácil limpeza, base drenante

### Para Playground
- **Recomendado**: BrassSoft Prime 40mm ou BrassSoft Luxo 40mm
- **Motivo**: Mais macias e confortáveis para crianças

### Para Varanda/Sacada
- **Pequenas**: BrassDecor 12mm ou 20mm (mais econômico)
- **Médias/Grandes**: BrassSoft Prime 20mm ou 30mm

### Para Eventos/Decoração
- **Verde**: BrassDecor 12mm (econômico)
- **Colorido**: BrassColor Decor 12mm ou BrassSoft Color 20mm

### Para Esporte
- **Futebol Society**: BrassSoccer Fibrilada ou Monofilamento
- **Tênis**: Brasstênis 20mm
- **Golfe**: BrassGolfe 12mm

## Informações Gerais

### Instalação
- **Sobre**: Concreto, cimento queimado, cerâmica, asfalto, pó de pedra
- **Fixação**: Cola de contato ou ganchos metálicos
- **Caimento**: Mínimo 1% para drenagem
- **Sim**: Instalamos na região metropolitana de Curitiba

### Limpeza
- Sabão neutro e água
- Lavadora de pressão com jato moderado
- Evitar calor direto (bitucas, solda)

### Proteções (todas as gramas)
- UV (não desbota)
- Antifúngica e antibacteriana
- Antichama (não propaga fogo)
